# Apple Store API

API REST para la gestión de una tienda virtual tipo Apple Store, desarrollada como proyecto académico.

## Tecnologías
- C#
- ASP.NET Core (.NET)
- SQL Server
- API REST

## Funcionalidades
- Autenticación de usuarios (login)
- Gestión de productos
- Carrito de compras
- Procesamiento de pedidos
- Integración con métodos de pago (simulado)

## Descripción
API desarrollada para manejar la lógica de negocio de una tienda virtual, incluyendo usuarios, productos, carrito de compras y pedidos.
